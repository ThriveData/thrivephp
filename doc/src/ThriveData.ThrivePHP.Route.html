    Class ThriveData\ThrivePHP\Route

    Defines the HTTP request to match and a callback to call if matched. Routes are registered with Router::register().

    Example:

    new Route(url: '{^/user/([0-9]+)/account$}', callback: '\ui\pages\user\account::routed');
new Route(url: '{^/api/v1/user/([0-9]+)/details$}', callback: '\api\v1\user\details::routed');

    Parameters

    $url  

    A regex to compare against the HTTP request URL. It is recommened to use the regex deliminators { and } as opposed to the usual / and / to avoid having to escape every forward slash.
    $callback  

    The method or function to call when the route is matched by Router.
    $method  

    A regex to compare against the HTTP method (e.g. GET, POST, etc.). By default matches any method.
    $host  

    A regex to compare against the HTTP host. By default matches any host.
    $id  

    A unique ID for identifying the exact Route among all registered routes. Probably only useful for debugging purposes.
